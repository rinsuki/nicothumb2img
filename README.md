# nicothumb2img

ニコニコ動画の https://ext.nicovideo.jp/thumb/sm9 を画像にするサービスです。

こうなります↓<br>
<img src="https://nicothumb2img.vercel.app/image/sm9" width="314" height="178">

## 使い方

```html
<img src="https://nicothumb2img.vercel.app/image/sm9" width="314" height="178">
```
以上！
- HiDPI な画像を返しているので、width と height の指定をしないとクソデカくなります
